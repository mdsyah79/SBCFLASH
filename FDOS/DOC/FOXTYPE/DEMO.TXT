FoxType is an advanced text viewer for DOS.

Main features are:

 * UTF-8 support
 * Word-wrapping
 * VESA 1280x1024 support
 * Ability to takes screenshots in BMP format

With FoxType you can view any UTF-8 text file. UTF-8 uses the Unicode table of characters, which is why it is the best way to guarantee that the characters in the text will not be misrepresented. Now, using UTF-8 files and FoxType, you can use several languages in the same text file!
